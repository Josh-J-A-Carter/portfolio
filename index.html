Chronodrive Video Game

This project was nominated for the Student's Slice category at The Pavs, for the 2025 ceremony.

Chronodrive is a pretty conventional, story-driven 2D platformer. I worked on it for around 2 months, as part of the COSC360 course at Otago University.

Through this project, I consolidated my skills with Unity 2D; particularly with C# and HLSL. I also learned quite a bit about pixel art, level design, and general project management.

Little Game Engine

In this project, I explored a number of very interesting technologies. Written in C++, I used SDL and OpenGL to create a simple 3D renderer from the ground up.

Some of the highlights have been learning more about 3D graphics & the OpenGL API, being able to apply C++ in medium-sized projects, and adding Emscripten to my toolkit for compiling to Web Assembly.

There's clearly a lot more I could do here, and I'm excited to keep working on this project!
