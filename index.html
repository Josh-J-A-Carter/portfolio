Projects

Chronodrive Video Game

This project was nominated for the Student's Slice category at The Pavs in the 2025 ceremony.

Chronodrive is a pretty conventional, story-driven 2D platformer. I worked on it for around 2 months, as part of the COSC360 course at Otago University.

Through this project, I consolidated my skills in Unity 2D, particularly with C# and HLSL. I also learned quite a bit about pixel art, level design, and general project management.

Little Game Engine

In this project, I explored a number of very interesting technologies. Written in C++, I used SDL and OpenGL to create a simple 3D renderer from the ground up.

Some of the highlights have been learning more about 3D graphics & the OpenGL API, being able to apply C++ to a decent-size project, and adding Emscripten to my toolkit for compiling to Web Assembly.

There's clearly a lot more I could do here, and I'm excited to keep working on this project!

Veracity Logic

A very interesting summer research project which involved combining the Event Calculus with the proposed Veracity Logic, for modelling the flow of trust in a network.

The back-end was written in Prolog, and is integrated with the proof assistant Coq to verify results at runtime - it was a lot of fun to set up! The front-end is written in simple HTML, CSS, & JS.
